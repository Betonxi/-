项目优势

围绕上市公司财报分析任务，FinChat 强调多智能体协同、结果可解释与交互自然化，兼顾科研展示与实际使用体验。

自然语言交互
无需掌握 SQL 或财报结构，直接用自然语言提问即可完成问数与深度分析。
多角色视角
支持投资者、企业管理者、监管机构等不同用户目标下的差异化分析表达。
结果可溯源
结合引用来源、共享黑板与执行链路，极大提升分析过程的透明性与可解释性。
多形式输出
同时支持文本回答、图表分析、风险提示与结构化报告生成，提升展示完整度。
Core Capabilities

从财报问数到综合分析
形成完整业务闭环

FinChat 不只回答单个指标问题，还可进一步完成企业运营评估、风险机会洞察与综合报告输出，高度契合科研展示和场景化落地需求。

"通过智能体编排，我们将原本孤立的数据查询，转化为了端到端的结构化洞察。"

智能问数
自动理解问题意图，生成查询路径，精准回答营收、利润、增长、对比等核心指标问题。
运营评估
围绕盈利能力、成长能力、运营效率、偿债能力和现金流质量，自动化输出诊断结果。
风险洞察
深度扫描并发现财务异常、现金流压力、经营波动和潜在机会，为决策提供预警与参考。
报告生成
无缝整合多智能体协同结果，一键输出更适合路演、答辩和成果展示的结构化分析报告。

推荐展示闭环

围绕固定演示链路组织答辩视频与现场展示，能够更清晰地体现系统从查数到决策支持的完整价值。

Step 1-2
选择公司与角色
在工作台选择医药生物行业上市公司，并切换为投资者、管理者或监管机构。
Step 3-4
问数与运营评估
先验证基础问数能力，再展示五维评分、雷达图与同业对比形成经营诊断。
Step 5-6
风险机会与报告
展示规则扫描与研报佐证，再生成角色化分析报告，体现系统的综合判断与建议能力。
Step 7-8
证据与执行链路
展示图表、引用来源、财报与研报证据、共享黑板和 Trace，强化可解释性与可信度。

多智能体系统架构

以 Orchestrator 为核心调度引擎，联动多个垂直场景专用智能体，高效完成财报分析任务的拆解、执行和汇总。

01
Orchestrator
充当系统大脑，统一负责任务意图识别、执行路径编排与最终结果整合。
02
Query Agent
面向底层数据库与专业知识库，精准完成数据抽取与自然语言答案生成。
03
Assessment Agent
运用多维度财务模型，自动化进行企业经营评分和结构化健康度诊断。
04
Risk Agent
深度挖掘数据关联，识别隐性风险与潜在商业机会，形成前瞻性预警。
05
Report Agent
融汇全局分析视角，一键生成兼具专业度与可读性的完整可视化报告。

应用场景

围绕核心财报数据，FinChat 提供千人千面的洞察视角，无缝覆盖多类高价值用户群体的决策支持需求。

投资者
快速判读标的公司核心盈利能力、未来增长趋势与关键隐患风险点，深度辅助投研分析与跨行业标的横向比较。
企业管理者
立足全局视角进行经营健康度诊断、业务短板精准识别与行业对标，有效辅助企业内部分析和战略运营优化。
监管机构
依托智能审查引擎，从异常指标波动、违规风险暴露与可疑财务信号识别角度，提供高效的辅助研判参考依据。
